# Mockup com IoT
- Repositório focado em lançar dados em tempo real dos sensores das 4 placas para o HiveMQ
- Recebimento de dados feito por tópicos, organizados por "nome_da_placa/tipo_sensor"
- Dados são enviados e recebidos de acordo com a necessidade do código
- Tratamento dos dados realizados para serem mostrados no APP do TremVida
